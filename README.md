## Features:

- Firebase auth:
    - Sign up
    - Sign in
    - Reset password
    - Update profile
    - Custom claims: Roles
    - Email confirmations
- Posts
    - Create new post with WYSIWYG editor
    - Edit post with WYSIWYG editor
    - Set publish state for each posts
    - Update changes in realtime
    - Tags
    - Categories
    - Comments
